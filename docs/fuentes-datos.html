Cómo extraer (sin scraping)

# El portal .gob.pe está tras WAF Huawei → exige User-Agent de navegador (sin él: HTTP 418)
UA="Mozilla/5.0 ... Chrome/124.0 Safari/537.36"
curl -A "$UA" -e "https://www.datosabiertos.gob.pe/" \
     -o E_2023.zip "https://www.datosabiertos.gob.pe/sites/default/files/E_2023.zip"

# Leer .dbf (UTF-8) → filtrar FENOMENO → agrupar por COD_UBIGEO → contrato JSON
import geopandas as gpd
gdf = gpd.read_file("E_2023.shp")
RAIN = {"LLUVIAS INTENSAS","INUNDACIÓN POR DESBORDE DE RIO","HUAYCOS",
        "DESLIZAMIENTO","INUNDACIÓN POR DESBORDE DE CANALES","EROSION FLUVIAL",
        "TEMPORALES (VIENTOS CON LLUVIAS)"}
NORTE = ("20","24","14","13")  # Piura, Tumbes, Lambayeque, La Libertad
f = gdf[gdf.FENOMENO.isin(RAIN) & gdf.COD_UBIGEO.str.startswith(NORTE)]
conteo = f.groupby("COD_UBIGEO").size()  # → { ubigeo, nombre, conteo, nivel, anios[] }

  • Espejo más estable (gob.pe / SENAMHI): 98 resultados — ideal para scraping del listado
  • Descarga directa de un PDF: https://enfen.imarpe.gob.pe/?wpdmdl=<ID> 200 · application/pdf
Estado vigente hoy (2026-06-26): ALERTA DE EL NIÑO COSTERO
Comunicado Oficial ENFEN N°11-2026 (15-jun-2026). Se proyecta hasta verano 2027.

Estado = campo discreto (región Niño 1+2): No Activo · Vigilancia de El Niño Costero · Alerta de El Niño Costero. (Nota Técnica ENFEN 02-2024.)

Periodicidad: ~quincenal (viernes) en periodo activo, mensual en calma, + extraordinarios. Numeración reinicia cada año. Sin API/JSON/RSS → solo HTML + PDF.

Para Vigía: descargar el PDF del último comunicado → 1 llamada real al modelo para resumirlo a 2-3 frases (coherente con el constitution). ⚠️ El token &ind=… de la URL expira: usar solo wpdmdl=<ID>.

3 · Límites distritales — la geometría (polígonos)

Shapefile IGN DESCARGA HOY

  • Página: datosabiertos.gob.pe/dataset/limites-departamentales
  • Descarga: https://www.datosabiertos.gob.pe/sites/default/files/DISTRITOS_LIMITES.zip · 18 MB 200 · application/zip

Escala 1:100.000 · referencial (sin vintage declarado) · SHP. Extraer: ogr2ogr/GeoPandas → GeoJSON → filtrar costa norte por prefijo ubigeo → simplificar para móvil.

Capa INEI Versión 2023 — para reconciliar ubigeo

  • Visor SDOT / Demarca Perú: geosdot.servicios.gob.pe/visor/ 200
  • Capa “Límite distrital — INEI Versión 2023” (cosecha vigente, “referencial no demarcatorio”)
⚠️ Riesgo del CONTEXT: divisiones de Piura post-2013 — Veintiséis de Octubre (Ley 29991, 2013) y La Unión. El IGN 1:100.000 puede no traerlos separados → validar ubigeos contra el catálogo INEI 2023.

Plan B open-source: github.com/juaneladio/peru-geojson (verificar que incluya los ubigeos nuevos).

Caveats operativos (antes del ETL)

  • WAF Huawei en todo *.gob.pe: manda User-Agent de navegador siempre (+ Referer en datosabiertos). Sin él → HTTP 418. El apex sin www. no resuelve.
  • SINPAD multi-año frágil: XLS rotos; solo bajan E_2018 y E_2023. Para el clímax basta 2023 (134 reg. costa norte). → valida el plan-B del CONTEXT (anclar ~15 distritos).
  • Encoding UTF-8 en los DBF (no latin1): INUNDACIÓN lleva tilde.
  • Límites referenciales (IGN e INEI), no demarcatorios — perfectos para choropleth.
  • ENFEN sin API: el estado cambia cada quincena; no cachear el estado a largo plazo (sí el resumen de IA del comunicado concreto).
